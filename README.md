# TiddlyWiki Plugins & Themes

My [TiddlyWiki](https://tiddlywiki.com/) plugins and themes.

## Install

To install:

```
yarn
```

## Develop

To develop:

```
yarn start
```

## Build

To build:

```
yarn build
```
